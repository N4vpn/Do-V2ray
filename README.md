# V2Ray One-Click Install Script for DigitalOcean VPS

This script installs and runs V2Ray (VLESS + WebSocket, no TLS) on port 8080 with the path `/TG-@n4vpn`.

Run the following command on your DigitalOcean VPS:

```bash
